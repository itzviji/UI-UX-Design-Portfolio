# 📖 About

Welcome to my **UI/UX Design Portfolio**!

This repository showcases the interface designs and prototypes I created while learning **Figma**. Every project represents a step in my design journey—from simple authentication screens to complete mobile applications and responsive website interfaces.

Each design helped me explore typography, layout composition, spacing, color systems, visual hierarchy, component reuse, and user-centered design principles.

---

# 🚀 Featured Projects

| Project | Description |
|----------|-------------|
| 🌿 **FloraCheck Mobile Application** | AI-powered plant disease detection mobile application designed as part of my IEEE research project. |
| 👗 **Fashion E-commerce Landing Page** | Modern responsive shopping website with promotional banners, product sections, testimonials, and footer. |
| 💊 **Wellness Onboarding UI** | Complete onboarding experience including authentication, dashboard, and personalized health questionnaire. |
| 🍕 **Food Ordering Menu UI** | Restaurant application featuring category-based navigation and reusable product cards. |
| 👚 **Fashion Authentication UI** | Modern login and registration interface with social authentication. |
| 📱 **Minimal Authentication Flow** | Beginner mobile sign-up and login interface focusing on clean layouts and typography. |
