# 👗 Fashion E-Commerce Landing Page

## 📖 Overview

This project is a beginner UI/UX web design created while learning **Figma**. The objective was to design a modern, responsive fashion e-commerce landing page that provides users with an engaging online shopping experience.

The website showcases a complete landing page including product categories, featured collections, promotional offers, testimonials, and a structured footer, following modern e-commerce design principles.

---

# 🎯 Design Objective

The primary goal of this project was to:

- Design a visually appealing fashion website
- Create a responsive landing page layout
- Improve product discoverability
- Build reusable UI sections
- Practice web interface design principles
